# Regras de Pontuação - Bolão do Hexa

Bem-vindo ao Bolão! Para que todos possam acompanhar e entender como o ranking é formado, abaixo explicamos detalhadamente como funciona o nosso sistema de pontuação para as partidas e para as apostas bônus (Bonus Bets).

---

## ⚽ Pontuação das Partidas

A pontuação de cada partida é calculada comparando o seu palpite com o resultado oficial do jogo. As regras são aplicadas na seguinte ordem (apenas a maior pontuação atingida é contabilizada):

### 1. Placar Exato (10 Pontos)

Você ganha 10 pontos se acertar exatamente o número de gols de ambas as equipes.

- **Exemplo**:
  - **Seu Palpite:** Brasil 2 x 0 Sérvia
  - **Resultado Oficial:** Brasil 2 x 0 Sérvia
  - **Pontos ganhos:** 10 pontos

### 2. Vencedor (ou Empate) + Acerto de um dos placares (7 Pontos)

Você ganha 7 pontos se acertar quem venceu a partida (ou se foi empate) **E** acertar a quantidade de gols de pelo menos uma das equipes.

- **Exemplo 1 (Vitória):**
  - **Seu Palpite:** Brasil 3 x 0 Sérvia
  - **Resultado Oficial:** Brasil 2 x 0 Sérvia
  - **Pontos ganhos:** 7 pontos _(Acertou a vitória do Brasil e os 0 gols da Sérvia)_
- **Exemplo 2 (Empate):**
  - **Seu Palpite:** Brasil 1 x 1 Suíça
  - **Resultado Oficial:** Brasil 2 x 2 Suíça
  - **Pontos ganhos:** 7 pontos _(Acertou que seria empate, mas errou o placar exato)_ _(Nota: Qualquer empate não exato cai na regra de 5 pontos abaixo, a menos que as regras considerem um lado acerto. No sistema, como empate exige que os dois sejam iguais, você não pode acertar "um dos lados" em um empate sem acertar o placar exato. Logo, empates não exatos valem 5 pontos)._
    _(Correção)_: Um exemplo real de 7 pontos seria: Palpite: Brasil 3 x 1. Resultado: Brasil 2 x 1 (Acerta o vencedor e o gol do time visitante).

### 3. Vencedor ou Empate (5 Pontos)

Você ganha 5 pontos se acertar apenas o resultado da partida (quem venceu ou se foi empate), mas errar a quantidade de gols de ambas as equipes.

- **Exemplo:**
  - **Seu Palpite:** Brasil 1 x 0 Sérvia
  - **Resultado Oficial:** Brasil 3 x 1 Sérvia
  - **Pontos ganhos:** 5 pontos _(Acertou a vitória do Brasil, mas errou ambos os placares)_

### 4. Acerto de um dos Placares (3 Pontos)

Você ganha 3 pontos se errar o resultado da partida (vencedor/empate), mas acertar os gols feitos por UMA das equipes.

- **Exemplo:**
  - **Seu Palpite:** Brasil 2 x 1 Camarões
  - **Resultado Oficial:** Brasil 0 x 1 Camarões
  - **Pontos ganhos:** 3 pontos _(Errou o vencedor, mas acertou que Camarões faria 1 gol)_

### 5. Nenhum Acerto (0 Pontos)

Quando seu palpite passa longe tanto do vencedor quanto dos placares das duas equipes.

- **Exemplo:**
  - **Seu Palpite:** Brasil 3 x 0 Camarões
  - **Resultado Oficial:** Brasil 0 x 1 Camarões
  - **Pontos ganhos:** 0 pontos

## 🏆 Apostas Bônus (Bonus Bets)

Além dos palpites em cada jogo, você pode dar palpites extras para o desfecho do campeonato! Essas apostas são computadas ao fim do torneio.

Os valores de cada aposta bônus são:

1. **Campeão: 50 Pontos**
   - Se você acertar a seleção que levantará a taça.
2. **Vice-Campeão: 25 Pontos**
   - Se você acertar a seleção que ficará em 2º lugar.
3. **Artilheiro: 70 Pontos**
   - Se você acertar o jogador que fará mais gols na competição.

_(Estes pontos serão somados à pontuação total das partidas para definir os grandes vencedores do Bolão!)_

Boa sorte e bons palpites!
